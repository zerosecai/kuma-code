# @kuma-code/skill-system

The skill pack runtime: load, retrieve, and auto-learn from skill packs.

This package powers Kuma Code's hybrid local+cloud architecture.
A skill pack is a 1GB modular knowledge bundle that lets small models
(0.1-3B) perform like big models on common tasks via a 2-level
table-of-contents lookup.

See `docs/SKILL_FORMAT.md` for the format specification.

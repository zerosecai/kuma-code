### Install directly

1. If you don't have VS Code installed, download it from [code.visualstudio.com](https://code.visualstudio.com/)
2. Then, you can click the button below to install Kuma Code directly from the VS Code Marketplace:

[![Install Kuma Code](https://raster.shields.io/badge/Install%20Kilo%20Code-F8F674?style=for-the-badge)](vscode:extension/kuma-code.kilo-code)

### Install from VS Code Marketplace

1. Open VS Code
2. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS) to open Extensions
3. Search for "Kuma Code"
4. Click **Install**

### Install via Command Line

```bash
code --install-extension kuma-code.kilo-code
```

### Verify Installation

After installation, you should see the Kuma Code icon ({% kiloCodeIcon /%}) in the Activity Bar on the left side of VS Code. Click it to open the Kuma Code panel.
